1. Перейти в конец строки
    <End>

2. Перейти в начало строки
    <Home>

3. Перейти в конец файла
    <Ctrl> + <End>

4. Перейти в начало файла
    <Ctrl> + <Home>

5. Перейти к строке с заданным номером
    <Ctrl> + <g>

6. Выделить слово перед курсором
    <Ctrl> + <Shift> + <LeftArrow>

7. Выделить слово после курсора
    <Ctrl> + <Shift> + <RightArrow>

8. Выделить весь текст от курсора до начала строки
    <Shift> + <Home>

9. Выделить весь текст от курсора до конца строки
    <Shift> + <End>

10. Выделить всё
    <Ctrl> + <a>

11. Копировать выделенный текст
    <Ctrl> + <c>

12. Вставить выделенный текст
    <Shift> + <v>

13. Поиск заданного слова в тексте
    <Ctrl> + <f>

14. Перейти к следующему слову в строке
    <Ctrl> + <RightArrow>

15. Перейти к предыдущему слову в строке
    <Ctrl> + <LeftArrow>

16. Перейти к следующей скобке (когда имеется пара скобок)
    <Ctrl> + <Shift> + <\>

17. Удалить строку
    <Ctrl> + <Shift> + <k>

18. Удалить слово перед курсором
    <Ctrl> + <BackSpace>

19. Удалить слово после курсора
    <Ctrl> + <Delete>

20. Увеличить количество отступов для выделенного фрагмента текста
    <Tab>

21. Уменьшить количество отступов для выделенного фрагмента текста
    <Shift> + <Tab>

22. Отменить предыдущую операцию (undo)
    <Ctrl> + <z>

23. Отменить отмену операции (redo)
    <Ctrl> + <y>

25. Увеличить текст (zoom in/zoom out)
    <Ctrl> + <+>

26. Уменьшить текст (zoom in/zoom out)
    <Ctrl> + <->

27. Открыть встроенный в VS Code терминал
    <Ctrl> + <Shift> + <~>

28. Закрыть встроенный в VS Code терминал
    <Ctrl> + <d>

29. скопировать текст в терминале
    <Ctrl> + <Insert>

30. вставить текст в терминал
    <Shift> + <Insert>

31. Очистить терминал:
    <Ctrl> + <l>

32. Прервать выполнение команды в терминале:
    <Ctrl> + <c>

33. Переместить строку (или несколько выделенных строк) на строчку выше
    <Alt> + <UpArrow>

34. Переместить строку (или несколько выделенных строк) на строчку ниже
    <Alt> + <DownArrow>

35. Найти файл:
    <Ctrl> + <p>

36. Закрыть вкладку:
    <Ctrl> + <w>

37. Поделить экран в текстовом редакторе VS Code надвое (split screen):
    <Ctrl> + <\>

38. Закомментировать выделенный фрагмент кода:
    <Ctrl> + </>

39. Открыть Explorer (боковую панель с папками и файлами):
    <Ctrl> + <Alt> + <e>

40. Закрыть Explorer (боковую панель с папками и файлами):
    <Ctrl> + <Shift> + <e>

41. Развернуть на весь экран (full screen mode):
    <F11>
