This is step-by-step instruction you have to follow in your day-to-day
work as developers (assuming you're familiar with Git basics already):

0. git checkout <your_branch_name>
1. git pull
2. write some code
3. git status
4. git add <filename>
5. git commit
6. Write commit message in text editor
7. Save & quit text editor
8. git push
9. go to github.com and make sure that new commits are there
