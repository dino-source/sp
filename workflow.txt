This is step-by-step instruction, you have to follow in your day-to-day work
as a developer taking into accout that you're familiar with Git basics already:

1. git pull
2. write some code
3. git status
3. git add <filename>
4. git commit
5. Write commit message in text editor
6. Save & quit text editor
7. git push
8. go to github.com and make sure that new commits are there
